MVP End-of-dungeon vouch + reputation database (TBC Anniversary 2.5.5)
Version: 0.2.8
------------------------------------------------------------------------

/mvp db
/mvp start
/mvp end
/mvp status
/mvp sync
/mvp sources


upon forming a group and entering a dungeon, addon will autotrack players 
and at the normal end of dungeon, pop up a vouching system to generate 
player reputation. Players vouched can be seen in "/mvp db" and when they 
join party or list themselves in LFG tool, you can view their reputation. 
Has optional good feedback comments and mandatory negative feedback comments. 
If a run ends prematurely, stop the mvp run by using "/mvp end". Sync's will 
be prompted by the user upon login and every 12 hours after on next login, 
or manually in the "/mvp db" window by pushing the sync button or typing "/mvp sync".
